Project Name : Vehicle Rental System.
Live URL : https://vercel.com/sabbir-rayhan-s-projects/vechicle-rental-system/9Axku2jdxvezMffEma5JiURR6Rus



Features : Theres are multiple operations present here.Customers and Admins Operations are different. And They are maintained by Authenticate Middleware.The main Features are :
    --> User authentication (Signup/Login with JWT)
    --> Role-based access control (Admin & Customer)
    --> Vehicle management (Add, Update, Delete, View)
    --> Booking system with date validation
    --> Automatic price calculation based on rental duration
    --> Vehicle availability management
    --> Admin can view all bookings
    --> Customer can view only their own bookings



Technology Stack
    --> We have used node.js,express.js, Typescript for the implementation. For Database we've used Neon.db. For frontend communicaiton we used Postman.Also use multiple npm packages like jwt,bcrypt,pg etc.
